# Repo txAdmin (Recipe Remote URL)

IMPORTANT:
- Le fichier YAML doit être à la RACINE du repo.
- Le dossier recipe_files doit être à la RACINE du repo (pas dedans un sous-dossier).
- Le repo doit être PUBLIC (sinon raw nécessite un token et txAdmin échoue souvent).
